In Python, data types define the kind of value a variable can hold. Every value in Python has a data type. Understanding data types is essential because it determines what operations you can perform and how Python will process your data.

Python's Main Data Types

Data Type Example Description Mutable
int 42 Whole numbers (positive or negative) No
float 3.14 Decimal numbers No
str "Hello" Text (sequence of characters) No
bool True, False Boolean truth values No
list [1, 2, 3] Ordered, changeable collection Yes
tuple (1, 2, 3) Ordered, unchangeable collection No
dict {"name": "Legend"} Key-value pairs (unordered) Yes
set {1, 2, 3} Unordered collection of unique items Yes
