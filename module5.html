Conditional statements allow your program to make decisions and execute different code blocks based on whether conditions are True or False. This is fundamental for creating dynamic, responsive programs that can handle different situations.

1. The if Statement

The most basic conditional statement. Code inside the if block only runs if the condition is True.

# Basic if statement syntax
if condition:
    # Code to run if condition is True

 2. The if...else Statement

            
Use else to specify code that should run when the if condition is False.

            
            
# if...else syntax
if condition:
    # Code if True
else:
    # Code if False

 3. The elif Statement

            
elif (short for "else if") lets you check multiple conditions in sequence.

            
            
# if...elif...else syntax
if condition1:
    # Code if condition1 is True
elif condition2:
    # Code if condition2 is True
else:
    # Code if all above are False

            
            

                
Conditional Flow

                
                
Python checks conditions in order until one is True, then skips the rest.

Important: Python will execute the first condition that evaluates to True and skip all subsequent elif and else blocks, even if they would also be True.

            

        

        
        
        

            
 4. Nested Conditionals

            
You can place if statements inside other if statements to create more complex logic.

            
            
# Nested if example
if outer_condition:
    if inner_condition:
        # Both conditions are True
    else:
        # Outer True, Inner False
else:
    # Outer condition False

 5. Logical Operators in Conditionals

                        Operator
                        Description
                        Example

                        or
                        Logical OR
