🧠 Module 2: Python Basics
🔹 1. What Are Variables?
Variables store data in memory. They are like labels for values you want to use later.
name = "Legend"
age = 19
height = 5.8
is_coder = True
🔎 2. Major Data Types in Python
These are the most common types Python variables can store:
  • str – Text: "Legend"
  • int – Whole numbers: 19
  • float – Decimals: 5.8
  • bool – True/False: True
  • list – Multiple items: [1, 2, 3]
  • tuple – Fixed items: (4, 5, 6)
  • dict – Key-value pairs: {"name": "Legend"}
  • set – Unique items: {1, 2, 3}
  • NoneType – Represents nothing: None
🧾 3. Naming Rules
  • Must begin with a letter or underscore
  • No spaces or special characters
  • Python is case-sensitive
valid_name = "Legend"
Invalid Name = ❌ # contains space
2score = ❌ # starts with number
🔹 4. Input & Output
input() gets user input, print() shows output.
username = input("Enter your name: ")
print("Welcome,", username)
💡 Remember: input() always gives you a string.
💬 5. Comments
Use comments to explain your code. Python ignores them when running.
# This is a single-line comment

"""
This is a
multi-line comment
"""
➗ 6. Math in Python
Python handles math like a calculator.
  • + Add
  • - Subtract
  • * Multiply
  • / Divide
  • // Floor Division
  • % Modulus (Remainder)
  • ** Power
a = 10
b = 3
print(a + b)  # 13
print(a ** b) # 1000
🧪 7. Practice Challenge
Create a program that asks for your name and age, then tells you the year you'll turn 100.
from datetime import date

name = input("Enter your name: ")
age = int(input("Enter your age: "))

year = date.today().year + (100 - age)
print(name, "will turn 100 in", year)
🏁 Recap
  • Variables store data you can reuse
  • You learned key data types in Python
  • input() + print() = basic interaction
  • Comments keep your code readable
  • Python can do math just like a calculator
