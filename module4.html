Operators are special symbols or keywords that perform operations on variables and values. Python provides various types of operators to perform calculations, comparisons, logical tests, and more. Mastering operators is essential for writing efficient and expressive Python code.

1. Arithmetic Operators

Used for mathematical operations between numeric values:

Operator Name Example Result
+ Addition 3 + 2 5
- Subtraction 5 - 2 3
* Multiplication 4 * 3 12
/ Division 10 / 2 5.0
// Floor Division 10 // 3 3
% Modulus (Remainder) 10 % 3 1
** Exponent 2 ** 3 8

Note: The / operator always returns a float. Use // for integer division.

 4. Logical Operators

            
Used to combine multiple conditions:

            
            

                
                    

                        Operator
                        Description
                        Example
                        Result
                    

                
                
                    

                        and
                        True if both conditions are true
                        x > 2 and x < 10
                        True if x is between 2 and 10
                    

                    

                        or
                        True if at least one condition is true
                        x < 5 or x > 10
                        True if x is less than 5 OR greater than 10
                    

                    

                        not
                        Reverses the result
                        not(x > 3)
                        True if x is NOT greater than 3
                    

                
            

            
            

                
Remember: Python uses short-circuit evaluation for logical operators. With and, if the first condition is false, the second won't be evaluated. With or, if the first is true, the second won't be evaluated.

 5. Identity & Membership Operators

            
            
 Identity Operators

            
Used to compare object identities (memory addresses):

            
            

                
                    

                        Operator
                        Description
                        Example
                    

                
                
                    

                        is
                        True if both variables are the same object
                        x is y
                    

                    

                        is not
                        True if both variables are not the same object
                        x is not y
                    

                
            

            
            

                
Note: is checks for identity (same object), while == checks for equality (same value).

            

            
            
 Membership Operators

            
Used to test if a value exists in a sequence (list, tuple, string, etc.):

            
            

                
                    

                        Operator
                        Description
                        Example
                    

                
                
                    

                        in
                        True if value exists in sequence
                        "a" in "apple"
                    

                    

                        not in
                        True if value doesn't exist in sequence
                        "z" not in "apple"

 Key Takeaways

            
    
                
  • Python provides various operators for different purposes: arithmetic, comparison, assignment, logical, identity, and membership.
    • 
                
  • Arithmetic operators (+, -, *, /, etc.) perform mathematical calculations.
    • 
                
  • Comparison operators (==, !=, >, etc.) compare values and return True or False.
    • 
                
  • Assignment operators (=, +=, -=, etc.) assign and update variable values.
    • 
                
  • Logical operators (and, or, not) combine multiple conditions.
    • 
                
  • Identity operators (is, is not) check if objects are the same.
    • 
                
  • Membership operators (in, not in) check if values exist in sequences.
    • 
                
  • Understanding operators is crucial for writing effective conditions and expressions in Python.
