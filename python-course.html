Introduction to Python

45 min Completed

What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages worldwide.

Python is used by major companies like Google, Facebook, Instagram, Netflix, and more!

Why Learn Python?

  • Beginner-friendly: Simple syntax that reads like English
  • Versatile: Used in web development, data science, AI, and more
  • Large community: Tons of resources and support available
  • High demand: One of the most sought-after skills in tech

Installing Python

To get started with Python, you'll need to install it on your computer:

  1. Visit python.org/downloads
  2. Download the latest Python installer for Windows
  3. Run the installer and check "Add Python to PATH"
  4. Complete the installation
  5. Verify by opening Command Prompt and typing python --version
  1. Mac usually comes with Python pre-installed
  2. Check version in Terminal with python3 --version
  3. To get the latest version, visit python.org/downloads
  4. Download and run the macOS installer

For Android devices, you can use these apps:

  • Pydroid 3 - Full Python IDE
  • QPython - Python for Android
  • Dcoder - Mobile code compiler

Most Linux distributions come with Python pre-installed.

To install the latest version:

sudo apt update
sudo apt install python3

Your First Python Program

Let's write the traditional "Hello, World!" program in Python:

# This is a simple Python program
print("Hello, Legend!")

Output:

Hello, Legend!

Try It Yourself

Modify the program to print your name instead of "Legend"

print("Hello, [Your Name]!")

  • It's used across many industries and applications
  • You can install Python on Windows, Mac, Linux, and even Android
