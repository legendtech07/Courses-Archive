üß† What is a Function?

A function is a block of reusable code that runs only when you call it. Instead of repeating code again and again, you define it once and reuse it anywhere.

üõ†Ô∏è Why Use Functions?

Functions help you:

  • üì¶ Break your code into smaller parts (modularity)
  • üîÑ Reuse the same logic multiple times
  • üîç Debug easier
  • üßº Keep your code clean and readable

üìö Types of Parameters

    
    

        

            Type
            Description
            Example
        

        

            1. Positional Parameters
            Values are passed in order
            subtract(5, 3)
        

        

            2. Keyword Arguments
            Pass values by name
            subtract(b=3, a=5)
        

        

            3. Default Parameters
            Set default values if not provided
            greet(name="Legend")
        

        

            4. Arbitrary Arguments (*args)
            For variable number of arguments
            total(1, 2, 3, 4)
        

        

            5. Keyword Arbitrary Arguments (**kwargs)
            For dynamic key-value pairs
            describe_person(name="Nazin", age=3000)
